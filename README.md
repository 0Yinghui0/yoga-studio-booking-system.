# Yoga and Mindfulness Studio Booking System

A robust, data driven web application built for a Yoga and Mindfulness Studio to manage course and class bookings. Built using Node.js, Express, NeDB, and Mustache templating, this project follows a strict Model View Controller (MVC) architecture.

## How to Run the Site

To run this application locally on your machine, please follow these steps:

1. Prerequisites: Ensure you have Node.js installed on your machine.
2. Install Dependencies: Open your terminal, navigate to the root folder of this project, and run the following command:
   npm install
3. Seed the Database: To generate the test courses and user accounts, run the seed script:
   node seed/seed.js
4. Start the Server: Once seeded, start the application by running:
   npm start
5. View the Site: Open your web browser and navigate to: http://localhost:3000

## Key Features Implemented

### Public Users
* Browse Courses: View all current and upcoming courses, including descriptions, locations, levels, and format.
* Search and Filter: Easily filter courses by Level (Beginner, Intermediate, Advanced) or search by keyword.
* View Sessions: See exact dates, times, and specific pricing for upcoming individual sessions.

### Registered Students
* Secure Authentication: Create an account and securely log in.
* Flexible Booking System: Choose to book an entire course block, or drop in and book specific individual sessions.
* Smart UI and Validation: The system actively prevents double bookings. If a full course is booked, individual session buttons intelligently disable.
* Personal Dashboard: View a complete history of upcoming and past bookings, including status updates if a class is modified by staff.

### Organisers (Staff)
* Staff Portal: Secure, role based dashboard strictly for users with Organiser privileges.
* Course Management (CRUD): Create, update, and delete main courses.
* Granular Session Control: Add specific dates, times, capacities, and custom pricing for individual classes within a course.
* Class Lists: Generate and view participant lists to see exactly who is attending a specific course or single session.
* Cascading Deletion Safety: If a course or session is deleted, the system preserves a snapshot in the student's history, marking their booking as cancelled rather than creating orphaned database records.
* User Management: View all registered users and safely delete accounts.
